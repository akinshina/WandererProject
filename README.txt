To get started with GitHub, you must do the following:

1.	Register an account on https://github.com/

2.	Install Git and Github.
	Git: http://git-scm.com/
	GitHub: https://windows.github.com/

3.	Fork the WandererProject repo

4.	Sync the repo through the GitHub app

5.	YOU'RE SET AND READY TO WORK
